Hamburgueria Fictícia

Este repositório contém o projeto de criação de uma landing page para uma hamburgueria fictícia, desenvolvido com o objetivo de demonstrar habilidades em desenvolvimento web e design.

Objetivos do Projeto

    Criar uma landing page atraente e responsiva para usuários de desktop e dispositivos móveis.
    Apresentar informações essenciais sobre a hamburgueria de forma clara e envolvente.
    Implementar funcionalidades interativas para aprimorar a experiência do usuário.

Funcionalidades

    Cabeçalho: Logo da hamburgueria e navegação simples.
    Seção de Apresentação: Imagem de destaque e uma breve descrição da hamburgueria.
    Menu: Destaques do menu com descrições e preços.
    Sobre Nós: Breve história da hamburgueria, missão e valores.
    Localização: Mapa interativo mostrando a localização da hamburgueria e informações de contato.
    Rodapé: Links para redes sociais e informações de contato.

Tecnologias Utilizadas

    Frontend: HTML, CSS, JavaScript.
    Outras Ferramentas: Git para controle de versão.
